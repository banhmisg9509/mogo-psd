# Photoshop basics
## Select Image
- Method 1
  - Move Tool ( Auto Select, Layer)
  - Select image -> Alt + Eye Icon
  - Image -> Trim
  - File -> Save for web (ctrl + alt + shift + s)
- Method 2
  - Move Tool (Auto Select, Layer)
  - Select image
  - Ctrl + C, Ctrl + N, Ctrl + V
  - File -> Save for web (ctrl + alt + shift + s)
